## 📦 Installation

    1. Installer un serveur web local php en fonction de votre environnement(WAMP, LAMP, XAMP, MAMP)

    2. cloner le repo :
        - Sur Windows
            * Ouvir le terminal et taper :
                ```
                cd C:\wamp64\www
                ```
            * Puis taper
                ```
                https://github.com/Yeti-Digital-Hub/Neo_Team_Projects.git
                ```
        - Sur Linux
            *Ouvir le terminal et taper :
                ```
                cd /var/www/html
                ```
            * Puis taper
                ```
                https://github.com/Yeti-Digital-Hub/Neo_Team_Projects.git
                ```
